# Vostok ML Research Lab

Laboratorio de investigación para el desarrollo y evaluación de técnicas de Machine Learning aplicadas a la restauración y análisis forense de audio.

## Objetivo

Explorar el uso de algoritmos de aprendizaje automático para complementar los sistemas DSP desarrollados en Vostok Restoration, investigando nuevas estrategias de:

* Clasificación de artefactos de audio.
* Detección inteligente de anomalías.
* Validación de detectores DSP.
* Segmentación espectral.
* Restauración asistida por Machine Learning.
* Sistemas híbridos DSP + ML.

## Filosofía del Proyecto

Este laboratorio tiene un carácter experimental y de investigación.

El objetivo no es reemplazar los detectores DSP existentes, sino evaluar si los modelos de Machine Learning pueden:

* Reducir falsos positivos.
* Reducir falsos negativos.
* Mejorar la clasificación de artefactos.
* Aumentar la confianza de las detecciones.
* Asistir futuros procesos de reparación automática.

## Relación con Vostok Restoration

Vostok ML Research Lab es un proyecto de investigación independiente que complementa el desarrollo de Vostok Restoration.

Las conclusiones obtenidas en este laboratorio podrán ser utilizadas en futuras versiones del motor de restauración, siempre que los resultados demuestren mejoras objetivas respecto a los métodos DSP tradicionales.
